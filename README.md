JS вводим в консоль.

В коде нужно будет изменить начальную ставку.

# Гарантия безопасности вашего профиля на сайте
Код полностью закомментирован. 
Для тех, кто слабо знаком с js, вы, можете поискать в JS 
"/send 76561198160884702 100" или что-то похожее...

```js
// Цитирую https://www.csgopolygon.com/support.php?language=ru

Чтобы отправить монеты другим пользователям, 
воспользуйтесь командой в чате: "/send [steam64id] [amount]".

Например, чтобы отправить 100 монет пользователю с steam64id 
76561198160884702, Вы должны написать команду 
в чат: "/send 76561198160884702 100" (пример).

Вы можете отправить монеты другому пользователю и другим 
способом, нажмите на изображение (аватар) пользователя правой 
кнопкой мыши, выберите пункт "Передать монеты", после укажите 
после появившегося текста в поле ввода сообщения сумму монет и 
отправьте.
```
